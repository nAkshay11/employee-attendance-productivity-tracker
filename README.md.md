**Employee Attendance \& Productivity Tracker**



**Project Overview**



This project analyzes employee attendance, productivity and departmental performance using Python, SQL and Power BI.



**Technologies Used**



\* Python

\* SQL

\* Power BI

\* Pandas

\* Excel/CSV



**Features**



\* Employee Productivity Analysis

\* Department-wise Performance Tracking

\* Attendance Analysis

\* Overtime Analysis

\* Salary and Efficiency Analysis



**Dashboard Insights**



\* Average Productivity Score

\* Department Distribution

\* Employee Performance Comparison

\* Department Productivity Trends

\* Average Salary KPI



**Tools Used**



\* MySQL Workbench

\* Python Pandas

\* Power BI Desktop

\* Excel
